# unibdf2hex-fork
Fork of Unifont unibdf2hex, Convert whole BDF font to Hex without hassle

for orignal source and authors, please refer to https://unifoundry.com/unifont/unifont-utilities.html

# Build
1. download original unifont utilites from link above
2. replace `unibdf2hex.c` with the one in this repository
3. `make bin`
